Project 1:

Hangman game


The Hangman Game project is a classic word-guessing game implemented in a digital format. The primary objective of this project is to create an interactive and entertaining game that challenges the player's word recognition and puzzle-solving skills. The game will provide an engaging user experience and test the player's ability to guess a hidden word correctly within a limited number of attempts.
